# Mini SOC using Wazuh

## Overview

This project demonstrates a simple SOC lab using:

- Wazuh Manager (Ubuntu)
- Wazuh Agent (Windows)

It simulates real-world SOC monitoring.

## Architecture

Ubuntu Server
   |
   |--- Wazuh Manager
   |
Windows Machine
   |
   |--- Wazuh Agent

## Features

✔ Log Monitoring  
✔ Intrusion Detection  
✔ File Integrity Monitoring  
✔ Brute Force Detection  
✔ Malware Detection

## Technologies

- Wazuh
- Ubuntu Server
- Windows
- Elastic Stack (optional)
